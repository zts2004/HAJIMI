# 💕 52行情话弹窗 PWA版本

一个可在手机浏览器中运行的52行情话弹窗应用，支持自动播放和爱心形状弹窗功能。

## ✨ 功能特点

- 📱 **PWA支持** - 可添加到手机主屏幕，离线使用
- 💌 **52句精选情话** - 每一句都充满爱意
- ▶️ **自动播放** - 自动连续弹窗显示情话
- 💖 **爱心形状** - 52个弹窗依次出现组成爱心形状
- 🎨 **多种颜色** - 88种柔和的背景颜色
- 📱 **移动端优化** - 完美适配手机屏幕
- 🔗 **链接分享** - 可通过链接在微信等社交软件中分享

## 🚀 使用方法

### 本地测试

1. 将所有文件放在一个文件夹中
2. 使用本地服务器打开（推荐使用VS Code的Live Server插件）
3. 或在浏览器中直接打开 `index.html`

### 部署到网络

#### 方法1：GitHub Pages（免费）

1. 在GitHub创建新仓库
2. 将所有文件上传到仓库
3. 在仓库设置中启用GitHub Pages
4. 选择主分支作为源
5. 获得类似 `https://你的用户名.github.io/仓库名` 的链接

#### 方法2：Vercel（免费，推荐）

1. 访问 [vercel.com](https://vercel.com)
2. 用GitHub账号登录
3. 导入你的GitHub仓库
4. 自动部署，获得链接

#### 方法3：Netlify（免费）

1. 访问 [netlify.com](https://netlify.com)
2. 拖拽文件夹或连接GitHub仓库
3. 自动部署，获得链接

### 生成短链接

部署后，可以使用短链接服务生成短链接：

- **推荐服务**：
  - [dwz.cn](https://dwz.cn) - 百度短网址
  - [url.cn](https://url.cn) - 腾讯短网址
  - [sina.lt](https://sina.lt) - 新浪短网址

## 📱 添加到主屏幕

### Android
1. 在Chrome浏览器中打开链接
2. 点击菜单（三个点）
3. 选择"添加到主屏幕"

### iOS (Safari)
1. 在Safari中打开链接
2. 点击分享按钮
3. 选择"添加到主屏幕"

## 🎯 功能说明

- **自动播放** - 点击后每隔0.8秒自动弹窗显示下一句情话
- **爱心形状** - 点击后52个弹窗依次出现（每0.3秒一个），排列成爱心形状，每个弹窗周围有冒泡爱心动画
- **ESC键** - 在浏览器中按ESC键可关闭所有弹窗并停止动画
- **关闭弹窗** - 点击每个弹窗右上角的×按钮可关闭单个弹窗

## 📝 文件说明

- `index.html` - 主HTML文件
- `app.js` - JavaScript核心逻辑
- `styles.css` - 样式文件
- `manifest.json` - PWA配置文件
- `sw.js` - Service Worker（离线支持）
- `README.md` - 说明文档

## 💡 注意事项

1. **HTTPS要求** - PWA功能需要HTTPS环境（本地localhost除外）
2. **图标文件** - 需要添加 `icon-192.png` 和 `icon-512.png` 图标文件（可选，不影响基本功能）
3. **浏览器兼容** - 推荐使用Chrome、Safari、Edge等现代浏览器

## 🎨 自定义

可以修改以下内容来自定义：

- `quotes` 数组 - 修改情话内容
- `popupColors` 数组 - 修改弹窗颜色
- `styles.css` - 修改样式和颜色主题
- `manifest.json` - 修改应用名称和图标
